# ecommerce-data-pipeline
E-commerce Data pipeline with sql (Bronze,Silver,Gold)
Project Overvieew 
This project demonstrates an end-to-end E-commerce Data Pipeline using Postgresql.

## Architecture 
Bronze - Raw Data
Silver - Cleaned Date 
Gold - Business Insights 

## Tech Stack 
-Postgresql
-Sql
-Power Bi (Comming next)
-Excel

## Project Structure 
-Sql/bronze - Row layer scripts 
-sql/silver -Cleaned data scripts 
-sql/gold - Business queries 
-screenshorts - Output visuals
